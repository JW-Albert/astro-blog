## Add the Sury APT Repository

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y lsb-release apt-transport-https ca-certificates wget

# Add Sury GPG key
wget -qO - https://packages.sury.org/php/apt.gpg | sudo apt-key add -

# Add Sury package source
echo "deb https://packages.sury.org/php/ $(lsb_release -sc) main" | sudo tee /etc/apt/sources.list.d/php.list

# Update package list
sudo apt update && sudo apt upgrade -y
```

## Install PHP 8.2

```bash
sudo apt install -y php8.2
```

## Used with Apache

```bash
sudo apt install -y libapache2-mod-php8.2
sudo systemctl restart apache2
```

## WordPress PHP Package Modules

```bash
sudo apt install -y php8.2-cli php8.2-fpm \
php8.2-mysql php8.2-xml php8.2-mbstring \
php8.2-curl php8.2-gd php8.2-zip php8.2-bcmath \
php-imagick php8.2-intl
```
